
## Ejercicio 2 - Ordenar lista de diccionarios

Ordena una lista de objetos según una clave usando sort y una función lambda.

**Entrada:**
```javascript
[
  { make: 'Nokia', model: 216, color: 'Black' },
  { make: 'Mi Max', model: 2, color: 'Gold' },
  { make: 'Samsung', model: 7, color: 'Blue' }
]
```

**Salida esperada:**
```javascript
[
  { make: 'Mi Max', model: 2, color: 'Gold' },
  { make: 'Samsung', model: 7, color: 'Blue' },
  { make: 'Nokia', model: 216, color: 'Black' }
]
```

## Ejercicio 3 - Matriz transpuesta

Calcula la transpuesta de una matriz usando map con funciones lambda.

**Entrada:**
```javascript
[
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]
```

**Salida esperada:**
```javascript
[
  [1, 4, 7],
  [2, 5, 8],
  [3, 6, 9]
]
```

## Ejercicio 4 - Eliminar elementos de una lista

Elimina los elementos indicados de una lista principal usando filter y una función lambda.

**Entrada:**
```javascript
Lista original: ['rojo', 'verde', 'azul', 'amarillo', 'gris', 'blanco', 'negro']
Elementos a borrar: ['amarillo', 'cafe', 'blanco']
```

**Salida esperada:**
```javascript
['rojo', 'verde', 'azul', 'gris', 'negro']
```
